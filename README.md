# 📇 eAlumina Contactos - Plataforma Colaborativa 

Plataforma de gestión de contactos integrada con Supabase. Permite agregar, importar y compartir contactos de forma colaborativa.

## Características

✅ **Agregar contactos manualmente** - Formulario simple (nombre, email, teléfono, notas)
✅ **Importar en lotes** - Soporta CSV, Excel, OpenDocument
✅ **Fusión inteligente** - Números sin nombres se unen con datos existentes
✅ **Deduplicación automática** - Evita duplicados por email o teléfono
✅ **Links compartibles** - Comparte un link para que otros carguen contactos
✅ **Búsqueda y filtrado** - Encuentra contactos rápidamente
✅ **API lista para agentes IA** - Acceso programático a todos los datos
✅ **Detección de incompletos** - Identifica contactos sin nombre, email o teléfono

## Instalación Local

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/contactos.git
cd contactos

# Abrir en navegador
open index.html
# o simplemente arrastra el archivo al navegador
```

## Configuración Supabase

La plataforma ya está conectada a Supabase. La clave API está configurada en `index.html`.

**Tabla requerida:**
```sql
CREATE TABLE public.contactos (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  created_at timestamp with time zone NOT NULL DEFAULT now(),
  nombre text NULL,
  email text NULL,
  telefono text NULL,
  estado text NULL
)
```

## Estrategias de Fusión

Al importar archivos, puedes elegir:

- **🧠 Inteligente (Recomendado)** - Completa campos vacíos sin sobrescribir datos existentes
- **⬇️ Usar nuevos datos** - Reemplaza todos los datos con los nuevos
- **⬆️ Mantener existentes** - Ignora contactos duplicados
- **❓ Preguntarme** - Muestra conflictos antes de fusionar
